# تطبيق إدارة عائلة آل سعدان - نسخة Netlify المحدثة

## نظرة عامة
- **الاسم**: تطبيق إدارة عائلة آل سعدان
- **الهدف**: منصة شاملة لإدارة شؤون العائلة والتواصل بين أفرادها
- **النوع**: تطبيق ويب تفاعلي مع حفظ البيانات محلياً (LocalStorage)
- **اللغة**: العربية مع دعم RTL كامل

## الميزات المكتملة ✅

### 🏠 الصفحة الرئيسية
- واجهة ترحيبية شاملة مع التنقل المتجاوب
- عرض تفاصيل مجلس الأسرة الحالي (الجيل الثالث)
- شبكة ميزات تفاعلية للانتقال السريع
- أدوات إدارة البيانات (تصدير وإعادة تعيين)

### 🌳 شجرة العائلة - إدارة متكاملة
- **عرض تفاعلي** للعائلة منظم حسب الأجيال (3 أجيال)
- **وضع التحرير** مع أزرار إدارة مرئية
- **إضافة أعضاء جدد** مع نموذج شامل:
  - الاسم الكامل، الأب، الجيل
  - رقم الهاتف والبريد الإلكتروني
  - ربط آباء مع اختيار ذكي
- **تحرير بيانات الأعضاء** مع ملء تلقائي للحقول
- **حذف الأعضاء** مع تأكيد أمان
- **حفظ تلقائي** في LocalStorage
- **أزرار إدارة منسقة** تظهر في وضع التحرير فقط

### 📅 إدارة الفعاليات
- عرض الفعاليات القادمة والمنتهية
- نظام الدعوات الذكي حسب مستوى العلاقة
- إضافة فعاليات جديدة مع تفاصيل كاملة
- تصفية وبحث في الفعاليات
- حفظ البيانات في LocalStorage

### 💡 نظام الاقتراحات - إدارة شاملة
- **إضافة اقتراحات جديدة** مع تفاصيل كاملة
- **تحرير الاقتراحات الموجودة**:
  - تحديث العنوان والمحتوى
  - تغيير التصنيف والأولوية
  - تحديث حالة الاقتراح (جديد، قيد المراجعة، تم التنفيذ، مرفوض)
- **حذف الاقتراحات** مع تأكيد أمان
- **عرض تفاصيل شاملة** في نافذة منبثقة
- **أزرار إدارة** (عرض، تحرير، حذف) لكل اقتراح
- **إحصائيات تحديثية** للاقتراحات
- **نظام تصفية** حسب الحالة
- **حفظ تلقائي** في LocalStorage

### 📚 المكتبة
- مكتبة محتوى متنوعة (وثائق، فيديو، صوتي)
- تصنيفات مختلفة (تعليمي، ثقافي، تاريخي)
- عرض شبكي منظم
- حفظ البيانات محلياً
